2014-08-01 Eizi TOYODA, Co-chair of WMO/CBS/IPET-MDRD

What is this

This is an implementation of WMO Core Metadata Profile to ISO 19115:2003 and ISO/TS 19139:2007 in the Schematron language (ISO/IEC 19757-3:2006).

Pure ISO 19139
